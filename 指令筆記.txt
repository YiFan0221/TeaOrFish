重要：
0.及時指令台
$heroku logs --tail

1.須新增的檔案
[runtime.txt]		運作所需的python版本
[requirements.txt]  	如果有import新套件版本記得登記到文件中
[Procfile	 ]	登記要執行的程式與種類

可用指令看有什麼套件來登記requirements.txt
$pip freeze

2.初始化
$git init

3.登入
$ heroku login

5.推送
$ git add .
$ git commit -am "make it better"
$ git push heroku master

6.重開機
$heroku restart

7.改時區
